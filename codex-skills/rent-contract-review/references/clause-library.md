# Tenant-oriented clause library for residential leases

Use only the clauses needed for the current contract. Adapt names, dates, payment timing, and local terminology.

## 1. Rental authority
甲方保证对出租房屋享有合法出租权，并已取得依法应当取得的授权、同意或批准。因房屋权属瑕疵、共有权人异议、授权不合法、房屋被查封或其他权利限制，导致乙方无法正常入住或继续使用房屋的，乙方有权解除合同，甲方应退还乙方已支付的全部押金及未履行期间租金，并赔偿乙方因此遭受的实际损失。

## 2. Handover annex
甲乙双方应于交房当日共同办理交接手续，并形成书面交接清单。交接清单应载明家具家电名称、品牌型号、数量、现状、已有瑕疵、水电燃气表底、相关账户余额、钥匙及门禁数量等内容。交房及退房时形成的照片、视频、抄表记录、门禁钥匙清单、物品清单均为本合同附件，与本合同正文具有同等法律效力。

## 3. Fee allocation
甲方负责结清交房日前产生的全部费用。租赁期间，乙方承担其实际使用期间发生的水费、电费、燃气费。物业费、宽带费、停车费及其他相关费用由双方约定如下：______。未经双方书面确认，甲方不得要求乙方承担本合同未明确约定由乙方承担的费用。

## 4. Repairs
租赁期间，房屋主体结构、原有装修及甲方提供的设施设备，包括但不限于水电燃气线路及管道、门锁、热水器、空调、照明设备、家具家电等，因非乙方原因发生故障、损坏或不能正常使用的，由甲方负责维修并承担费用。甲方在接到乙方维修通知后应及时处理；甲方逾期未维修并影响乙方正常居住使用的，乙方有权自行维修，相关合理费用凭有效票据由甲方承担，乙方有权向甲方追偿或在应付租金中抵扣。

## 5. Deposit refund
押金仅用于抵扣乙方实际欠付费用或乙方因故意、过失造成的实际损失。房屋及附属设施设备因正常居住、合理使用产生的自然损耗、折旧、老化，不属于押金扣除范围。租赁期满或合同依法解除后，乙方完成交房、返还钥匙门禁并结清应由乙方承担的费用后，甲方应于三个工作日内无息退还剩余押金。

## 6. Tenant termination right
有下列情形之一的，乙方有权解除合同，且不承担违约责任：甲方不具备合法出租权；房屋存在权属争议、被查封、被限制处分；房屋存在安全隐患、质量问题或其他严重影响正常居住使用的情形；甲方拒不履行维修义务，致使乙方无法正常居住使用；甲方无正当理由提前收回房屋；其他因甲方严重违约导致合同目的无法实现的情形。乙方依据本条解除合同的，甲方应退还全部押金及未履行期间租金，并赔偿乙方因此遭受的实际损失。

## 7. Landlord breach
甲方无正当理由提前解除合同、擅自收回房屋、阻碍乙方正常使用房屋，或存在其他严重违约行为的，应退还乙方未履行期间租金及全部押金，并赔偿乙方因此遭受的合理搬迁费、临时安置费及其他实际损失。

## 8. Entry, sale, renewal
非因紧急情况，甲方进入房屋前应提前通知乙方并征得乙方同意。甲方出售房屋的，不影响本租赁合同在租赁期限内继续履行。在同等条件下，乙方享有优先承租权。
