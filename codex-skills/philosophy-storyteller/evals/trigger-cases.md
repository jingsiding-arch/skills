# Trigger Cases

Use this skill when:

- “把这句话写成一个小故事：短暂的东西也可以是真的。”
- “这个道理能不能写成类似夏令营效应那种对话？”
- “边界感不是疏远，是让关系更长久。帮我讲成故事。”
- “写一个哲理小故事，核心是不要把火柴要求成太阳。”
- “把‘没有结果不代表没有意义’写成深夜聊天感。”

Do not use this skill when:

- “解释一下夏令营效应是什么。”
- “写一篇 3000 字哲学论文。”
- “帮我写一个品牌广告故事。”
- “模仿某某作家的风格写一篇小说。”
- “给我列 10 个心理学概念。”

Expected routing:

- Story transformation requests: yes.
- Pure explanation requests: no, unless the user asks to explain through a story.
- Long fiction requests: no, unless the user explicitly wants the same dialogue-philosophy format.
